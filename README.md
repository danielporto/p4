# P4 - Pequeno Processador Pedagógico com Pipeline

Para instalar os pacotes:

`npm install`

Para gerar a versão web:

`node web.js`

## Versão em container:
Para gerar os ficheiros para o docker:

`build_web.sh`

depois para correr:

`docker-compose up`

Abra no navegador: http://localhost
